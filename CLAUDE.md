# CLAUDE.md

This file provides guidance to Claude Code (claude.ai/code) when working with code in this repository.

## Project Context

This is a document repository for **Instituto Isanté / Grupo Silvestre Santé**, a Brazilian health and social-impact organization. The working directory is not a software project — it contains institutional documents in Portuguese (PDFs and Word files).

## Document Collection (`CLODE/`)

| File | Purpose |
|------|---------|
| `Apresentação Instituto Isanté (3).pdf` | Institutional presentation of Instituto Isanté |
| `Projeto_Amazonia_Sem_Fronteiras_24_meses_FINAL_04062026.pdf` | Final version of the "Amazônia Sem Fronteiras" 24-month project (finalized 2026-06-04) |
| `Portfolio de Investimentos em Projetos Silvestre Sante.docx` | Investment portfolio across Silvestre Santé projects |

| `Publicações e Linha do Tempo Cientifica Grupo Silvestre Santé.docx` | Scientific publications and research timeline |

## Working with These Documents

- PDF files cannot be read with the standard `Read` tool unless `pdftoppm` is installed. Use the user's own document tooling or request content be pasted in.
- Word (`.docx`) files can sometimes be read as raw XML; content may be partially legible via `Read`, though formatting will be stripped.
- All documents are in **Brazilian Portuguese** — respond in Portuguese when asked to draft, edit, or summarize content from them.
